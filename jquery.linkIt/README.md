#LinkIt jQuery Plugin#

**LinkIt** is a simple jQuery plugin that allows you to attach links to elements.

##Version##
0.3.0

##Usage##
    $('span').linkIt({
      href: "http://www.example.com",
      text: "Click here",
      target: '_blank'
    });

##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
